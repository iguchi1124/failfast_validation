# FailfastValidation
Failfast mode for ActiveModel::EachValidator

## Usage
```rb
class YourModel
  include ActiveModel::Model
  include ActiveModel::FailfastValidation
end
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'failfast_validation'
```

And then execute:
```bash
$ bundle
```
